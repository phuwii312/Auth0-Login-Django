# Auth0-Login-Django
Adding login feature using Auth0 on sample code
sample tutorial found on this website https://auth0.com/docs/quickstart/webapp/django/interactive

Please replace in the .env file your own ClientID, Client Secret and Domain (which can all be created on Auth0 website when creating your own application)
